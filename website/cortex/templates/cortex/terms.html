4 Premium Features & Access

Cortex provides certain advanced features that are available exclusively through an active subscription. These features enhance the AI-assisted development experience beyond standard code generation and editing.

4.1 Subscription-Only Features

🔒
Web Search and OCR / Image Recognition are subscription-only features. These capabilities require an active Pro subscription to function. They are not available on the Free plan or to BYOK-only users.
  • Web Search — enables the AI to search the internet in real-time for up-to-date documentation, library versions, API references, and solutions to recent technical issues.
  • OCR / Image Recognition — enables the AI to process, analyze, and extract text from images, screenshots, diagrams, and PDF documents submitted by the user.

4.2 BYOK Provider Support

When using a BYOK (Bring Your Own Key) configuration with an active subscription:

  • If your chosen BYOK provider natively supports web search or OCR capabilities (e.g., certain OpenAI or Qwen models with vision/search APIs), those features will function through your own API key.
  • If your chosen BYOK provider does not natively support web search or OCR (e.g., DeepSeek), Cortex will route these specific requests through its own provider infrastructure as a fallback, consuming your subscription credits.
  • This fallback mechanism ensures that premium features work seamlessly regardless of which model you have selected, provided you have an active subscription with available credits.

4.3 Without a Subscription

Without an active subscription, BYOK users retain access to their chosen AI provider for standard coding tasks at no cost from Cortex. However, web search and OCR features are subscription-only and will not function — even if the BYOK model theoretically supports those capabilities.
