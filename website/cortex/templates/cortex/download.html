Common questions

Download FAQ

Is Cortex free?

The IDE is free to download. To use AI features, you need a subscription ($10/mo) which includes platform services (OCR, embeddings, web search). You also bring your own API keys for LLM providers — you pay them directly.

What about my API keys?

Your keys are stored locally in Windows Credential Manager. They never leave your machine and are never sent to any Cortex server. You pay the LLM providers directly — Cortex doesn't mark up or resell tokens.

Can I use it offline?

The IDE itself works offline for code editing. AI agent features require an internet connection to reach your chosen provider's API. The $10/mo subscription covers platform services (OCR, embeddings, web search) which also require internet.

Does it work on Mac or Linux?

Currently Cortex is Windows-only. Mac and Linux support is on the roadmap.

Your keys stay local

Cortex is bring-your-own-key — your LLM requests go directly from your machine to the provider over TLS. Your prompts, code, and credentials never touch our infrastructure. The $10/mo subscription covers platform services (OCR, embeddings, web search) which are proxied through Cortex servers.
