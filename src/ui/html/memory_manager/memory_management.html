Agentic Loop Mode

Controls whether the AI may run an autonomous verified loop for iterative problem-solving. In Normal Mode, the AI can only work one task at a time and cannot start a loop. In Loop Mode, the AI may automatically repeat fix → verify → revise until the goal passes its verification gates or a limit is reached.

Allows Cortex to run multi-iteration agent loops with automated verification. Requires explicit goal and success criteria.

How the agentic loop works

The AI may now start an autonomous fix → verify → revise loop toward a goal, iterating on its own until the goal passes verification or a limit is reached. Set clear success criteria and allow appropriate budgets (iterations, tokens, USD).
